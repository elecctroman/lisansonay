Codex Pro brings a polished WooWallet admin experience with dashboards, wallet tools, and My Account enhancements.

== Description ==

* Dashboard with wallet adapter charts powered by Chart.js.
* Wallet management screen to inspect balances via WooWallet.
* Modern My Account endpoint with responsive cards.

== Installation ==

1. Upload the plugin zip to your WordPress site and activate it.
2. Ensure WooWallet (TeraWallet) is active for wallet features.
3. Visit Codex Pro → Settings to configure the My Account endpoint.

== Frequently Asked Questions ==

= Do I need WooWallet? =
Yes, the plugin extends WooWallet/TeraWallet. Without it, wallet features will be hidden.
